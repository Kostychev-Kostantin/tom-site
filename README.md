Top of Mind Labs — Starter site

This is a minimal static site scaffold that mirrors the structure of an enterprise consulting website (header, hero, services, industries, insights, contact CTA, footer).

How to view locally:

1. Open `index.html` in your browser (double-click or right-click -> Open With).
2. Or run a simple static server (Python):

```bash
python -m http.server 8000
# then open http://localhost:8000
```

Files added:

- `index.html` — main page
- `css/styles.css` — styles
- `assets/logo.svg`, `assets/hero.svg` — simple SVG placeholders
